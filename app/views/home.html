Welcome to MINIVAN
closed beta

What is this? MiniVan, as "mini visual analyser of networks", is an online tool for publishing and exploring networks. It is currently in closed beta. You can use this interface but please do not share its URL. At this point, the developers already have a huge road map, need specific feedback, but not a massive flow of issues or missing features that they are already dealing with.

How to use. You can upload a GEXF network in an interface named "Maker" and tune publication settings. You will get a "data bundle" containing your network, its metadata (title, description...) and publication settings. You need to host this bundle somewhere online, for instance on GitHub. You then have to pass the data bundle's URL to MiniVan, in order to explore your network.

1. MiniVan Maker. Upload a GEXF network, tune settings, get a data bundle.

2. Get the URL. Give your data bundle's URL, get a MiniVan URL. You need to host the data bundle somewhere.

3. Explore your network. Get the URL on step 2, or try a demo network.

4. Feedback. Help us by providing feedback. Use the GitHub issues (you need an account).

Source code. The project is free and open source and hosted on GitHub.
